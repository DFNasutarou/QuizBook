# 事実確認機能の使い方

クイズ作成時にClaude.aiを使って事実確認を行う方法です。

## 🚀 使い方（2ステップ）

### ステップ1: Chrome拡張機能をインストール

1. `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」をON
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. `chrome-extension` フォルダを選択

詳細は `chrome-extension/README.md` を参照してください。

### ステップ2: 事実確認を実行

1. **問題編集タブ** で問題と答えを入力
2. 「🔍 Claude.ai で事実確認」ボタンをクリック
3. **完了！** 自動で送信されます

## 🎯 何が起こるか

```
1. ボタンをクリック
   ↓
2. Claude.aiが新しいタブで開く
   ↓
3. 1〜2秒後、自動で質問が入力される
   ↓
4. 自動で送信される
   ↓
5. Claudeが事実確認の結果を返す
```

**手動操作は一切不要です！**

## 📊 Claudeが確認すること

1. **答えの正確性** - 事実として正しいか
2. **問題文の明確性** - 曖昧な表現がないか
3. **追加の関連情報** - 知っておくべき情報
4. **問題としての適切性** - 難易度や表現

## ❓ トラブルシューティング

### Q: 自動送信されない

**確認事項:**
1. Chrome拡張機能がインストールされているか
2. Claude.aiのコンソール（F12）でエラーが出ていないか
3. Claude.aiにログインしているか

**解決策:**
1. `chrome://extensions/` で拡張機能を再読み込み
2. Claude.aiのページをリロード
3. 再度ボタンをクリック

### Q: 拡張機能がインストールできない

**エラー:** 「Could not load icon」
→ 既に修正済みです。最新の`manifest.json`を使用してください。

**エラー:** 「マニフェストファイルがありません」
→ 正しいフォルダ（`chrome-extension`）を選択してください。

### Q: Claude.aiのUIが変わって動作しない

**対処法:**
Claude.aiがUIを更新した可能性があります。
`chrome-extension/content.js` の修正が必要な場合があります。

## 🔧 高度な使い方

### 連続して複数の問題を確認

1. 問題1を確認（自動送信）
2. 回答を読む
3. タブを閉じる
4. 問題2を確認（自動送信）
5. 繰り返し

### 1つのチャットで複数確認

- タブを閉じずに、次の問題を確認
- 同じチャットセッションで連続して質問できます
