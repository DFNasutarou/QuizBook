# Chrome拡張機能インストールガイド

クイズ管理ツールからClaude.aiへの自動入力・送信を実現するChrome拡張機能です。

## 🎯 機能

- ✅ **完全自動化** - ボタン1クリックで質問を自動送信
- ✅ **手動操作不要** - 貼り付け・送信が全自動
- ✅ **高速** - Claude.aiが開いて1〜2秒で送信完了
- ✅ **完全無料** - 追加コスト一切なし

## 📦 インストール手順

### ステップ1: 拡張機能フォルダを確認

現在いる場所: `D:\project\github\AC_WEB_DFN\quiz\tools\chrome-extension\`

以下のファイルがあることを確認してください：
- `manifest.json`
- `content.js`
- `background.js`
- `popup.html`
- `icon16.png` / `icon48.png` / `icon128.png` (オプション)

### ステップ2: Chromeの拡張機能ページを開く

1. Chromeを開く
2. アドレスバーに `chrome://extensions/` と入力してEnter
3. または、メニュー → その他のツール → 拡張機能

### ステップ3: デベロッパーモードを有効化

右上の「デベロッパーモード」トグルをONにする

### ステップ4: 拡張機能を読み込む

1. 「パッケージ化されていない拡張機能を読み込む」をクリック
2. フォルダ選択ダイアログが開く
3. `chrome-extension` フォルダを選択
   ```
   D:\project\github\AC_WEB_DFN\quiz\tools\chrome-extension
   ```
4. 「フォルダの選択」をクリック

### ステップ5: インストール完了

拡張機能一覧に「クイズ事実確認 自動入力」が表示されれば成功！

## 🔧 トラブルシューティング

### 拡張機能が読み込めない

**エラー**: 「マニフェストファイルがありません」

**解決策**:
- 正しいフォルダ（`chrome-extension`）を選択しているか確認
- `manifest.json` が存在するか確認

### 自動入力されない

**原因**: Claude.aiのUI構造が変更された可能性

**解決策**:
1. ブラウザのコンソールを開く（F12キー）
2. Console タブを確認
3. エラーメッセージを確認
4. `content.js` の修正が必要な場合があります

### 送信ボタンが押されない

**フォールバック**:
- 入力は自動で行われます
- 手動でEnterキーを押してください

### アイコンが表示されない

**問題なし**:
- アイコンがなくても機能は動作します
- 必要に応じて `icon16.png` などを追加してください

## 🎨 アイコンの追加（オプション）

アイコン画像を追加する場合：

1. 16x16, 48x48, 128x128 のPNG画像を用意
2. `icon16.png`, `icon48.png`, `icon128.png` として保存
3. `chrome-extension` フォルダに配置
4. 拡張機能をリロード

または、以下のサイトで無料アイコンを探せます：
- https://www.flaticon.com/
- https://icons8.com/

## 🔄 更新方法

拡張機能のコードを修正した場合：

1. `chrome://extensions/` を開く
2. 拡張機能カードの「再読み込み」ボタンをクリック
3. Claude.aiのページをリロード

## 🗑️ アンインストール

1. `chrome://extensions/` を開く
2. 拡張機能カードの「削除」をクリック

## 🔐 セキュリティとプライバシー

- **ローカルのみ**: この拡張機能はローカル環境でのみ動作
- **必要最小限の権限**: claude.ai へのアクセスのみ
- **データ送信なし**: 外部サーバーにデータを送信しません
- **オープンソース**: コードは全て確認可能

## ⚠️ 注意事項

- **Claude.aiのアカウントが必要**: ログインしていない場合は手動でログインしてください
- **UI変更に注意**: Claude.aiがUIを更新すると動作しなくなる可能性があります
- **非公式**: この拡張機能はAnthropicの公式ツールではありません
