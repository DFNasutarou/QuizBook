# 早押しクイズ管理ツール

友達に早押しクイズを出題するためのWebアプリケーションです。ブラウザで動作し、インストール不要で使えます。

## 機能

### 基本機能
- ✅ クイズ問題の新規作成・編集・削除
- ✅ 問題集の管理（複数の問題集を作成・管理可能）
- ✅ 問題文・答えの色付け機能
- ✅ ふりがな機能（漢字の読み方を括弧で表示）
- ✅ ジャンル・難易度・タグによる問題の分類
- ✅ 全体のフォントサイズ調整機能

### 出題機能
- ✅ ランダム出題機能
- ✅ 前の問題・次の問題への移動
- ✅ 答えの表示・非表示切り替え
- ✅ 複数問題集の組み合わせ出題
- ✅ ジャンル・タグ・難易度によるフィルター機能

### データ管理機能
- ✅ JSON形式での問題集保存・読み込み
- ✅ CSV形式でのインポート・エクスポート
- ✅ 既存問題集の自動整形機能

## バッチ運用

## インストール・使用方法

### 使い方

1. `index.html` をブラウザで開く
   - ファイルをダブルクリック、または
   - ブラウザにドラッグ&ドロップ

2. すぐに使い始められます！インストール不要です。

### 推奨ブラウザ

- Google Chrome (推奨)
- Microsoft Edge
- Firefox
- Safari

### ローカルサーバーで実行する場合（オプション）

```bash
# Pythonの簡易サーバーを使う場合
cd quiz/tools
python -m http.server 8000

# ブラウザで http://localhost:8000 を開く
```

## 使い方

### 1. 起動
- `index.html` をブラウザで開く

### 2. 問題集の管理

#### 新規問題集の作成
1. 「問題集管理」タブを開く
2. 「新規作成」ボタンをクリック
3. 問題集名を入力

#### 既存問題集の読み込み
1. メニューバーの「ファイル」→「問題集を開く」
2. JSONファイルを選択

#### CSVファイルのインポート
1. メニューバーの「ファイル」→「CSVをインポート」
2. CSVファイルを選択（形式は sample_quiz_format.csv を参照）

### 3. 問題の作成・編集

#### 新規問題の作成
1. 問題集を選択
2. 「新規問題」ボタンをクリック
3. 「問題編集」タブで内容を入力
4. 「保存」ボタンをクリック

#### 文字装飾機能
- **色付け**: テキストを選択して「色を付ける」ボタンをクリック
- **ふりがな**: 漢字を選択して「ふりがなを付ける」ボタンをクリック
  - 例: `漢字(かんじ)` の形式で入力

### 4. 出題機能

#### 出題の開始
1. 「出題」タブを開く
2. 問題集を選択
3. フィルター条件を設定（オプション）
4. 「出題開始」ボタンをクリック

#### 出題中の操作
- **前の問題**: 前の問題に戻る
- **次の問題**: 次の問題に進む
- **答えを表示/隠す**: 答えの表示を切り替え
- **ランダム**: ランダムに問題を選択
- **出題終了**: 出題を終了

#### フィルター機能
- **ジャンル**: 特定のジャンルの問題のみ出題
- **難易度**: 易・中・難から選択
- **タグ**: 特定のタグが付いた問題のみ出題

### 5. 設定

#### フォントサイズ調整
- 「設定」タブまたはメニューバーの「表示」から調整可能
- 文字が読めない場合に便利

## CSVファイル形式

インポート用CSVファイルの形式例：

```csv
問題文,答え,メモ,ジャンル,難易度,タグ
日本の首都はどこでしょう？,東京,基本的な地理問題です,地理,易,"地理, ベタ問, 基本"
「吾輩は猫である」の作者は誰でしょう？,夏目漱石(なつめそうせき),明治時代の代表的作家,文学,中,"文学, 作品名, 人名"
```

### フィールド説明
- **問題文**: 出題する問題文
- **答え**: 問題の答え
- **メモ**: 補足情報（オプション）
- **ジャンル**: 問題のジャンル（オプション）
- **難易度**: 易・中・難から選択（オプション、デフォルト: 中）
- **タグ**: カンマ区切りのタグ（オプション）

## 既存データの整形

既存のCSVファイル（Shift_JISエンコード）を自動的に整形してタグ・難易度を追加します。

```bash
cd scripts
python3 format_existing_data.py
```

### 整形機能
- 文字エンコーディングの自動変換（Shift_JIS → UTF-8）
- ふりがな形式の正規化
- ジャンルの自動推定
- 難易度の自動推定
- タグの自動生成

## データ保存について

- 問題集データはブラウザの **LocalStorage** に自動保存されます
- 「保存」ボタンでJSONファイルとしてダウンロード可能
- 「読み込み」ボタンでJSONファイルから復元可能
- CSVエクスポート機能で他のツールとの連携も可能

### 注意事項

- ブラウザのキャッシュをクリアするとデータが消える可能性があります
- 定期的にJSONファイルとしてバックアップすることを推奨します
- プライベートブラウジング（シークレットモード）ではデータが保存されません

## トラブルシューティング

### データが保存されない
- プライベートブラウジングモードを使用していないか確認
- ブラウザの設定でLocalStorageが有効になっているか確認
- 定期的に「保存」ボタンでJSONファイルにバックアップ

### 文字化けする場合
- CSVファイルのエンコーディングを確認
- UTF-8で保存されていることを確認

### フォントが小さい場合
- 「設定」タブでフォントサイズを調整できます

### ブラウザ間でデータを共有したい
- 「保存」ボタンでJSONファイルをダウンロード
- 別のブラウザで「読み込み」ボタンから復元
